This is the Rmd file with the requested datable that shows the average
mile per gallon of each car make in the mtcars dataset.

The code I used below highlights how I thought about answering this
problem.

First, I used regular expressions to extract all car makes from the car
models in the dataset.

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{mtcars_avg <-}\StringTok{ }\NormalTok{mtcars }\OperatorTok{%>%}\StringTok{ }
\StringTok{  }\KeywordTok{rownames_to_column}\NormalTok{(}\DataTypeTok{var =} \StringTok{"car"}\NormalTok{) }\OperatorTok{%>%}\StringTok{ }
\StringTok{  }\KeywordTok{mutate}\NormalTok{(}\DataTypeTok{make =} \KeywordTok{str_extract}\NormalTok{(car,}\StringTok{"[A-Z][a-z]+"}\NormalTok{))}
\end{Highlighting}
\end{Shaded}

Next, I grouped my previously saved table by make, and calculated the
mean mpg for each make using the tapply function in the apply family.

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{cars <-}\StringTok{ }\KeywordTok{data.frame}\NormalTok{(}\KeywordTok{tapply}\NormalTok{(mtcars_avg}\OperatorTok{$}\NormalTok{mpg, mtcars_avg}\OperatorTok{$}\NormalTok{make,mean)) }\OperatorTok{%>%}\StringTok{ }
\StringTok{  }\KeywordTok{rownames_to_column}\NormalTok{(}\DataTypeTok{var =} \StringTok{"Make"}\NormalTok{) }\OperatorTok{%>%}\StringTok{ }
\StringTok{  }\KeywordTok{rename}\NormalTok{(}\DataTypeTok{mpg =} \StringTok{"tapply.mtcars_avg.mpg..mtcars_avg.make..mean."}\NormalTok{) }\OperatorTok{%>%}\StringTok{ }
\StringTok{  }\KeywordTok{mutate}\NormalTok{(}\DataTypeTok{mpg =} \KeywordTok{round}\NormalTok{(mpg, }\DecValTok{2}\NormalTok{))}
\end{Highlighting}
\end{Shaded}

Finally, I used knitr's kable function to clean up the table for
presentation.

\begin{Shaded}
\begin{Highlighting}[]
\KeywordTok{kable}\NormalTok{(}\KeywordTok{list}\NormalTok{(cars[}\DecValTok{1}\OperatorTok{:}\DecValTok{11}\NormalTok{,], cars[}\DecValTok{12}\OperatorTok{:}\DecValTok{22}\NormalTok{,]),}
      \DataTypeTok{caption =} \StringTok{"Average miles per gallon for each car make"}\NormalTok{,}
      \DataTypeTok{booktabs =} \OtherTok{TRUE}\NormalTok{)}
\end{Highlighting}
\end{Shaded}
